# 🌦️ Weather App

A responsive weather app built with React that displays real-time weather data using the OpenWeather API.

## 🚀 Features

- Display real-time weather data for Gaza 🌍  
- Multi-language support using i18next 🌐  
- Clean and responsive UI for all screen sizes 📱💻  
- Date and time handling with Day.js ⏱️  

---

## 🛠️ Technologies Used

- React  
- Material UI (MUI)  
- Axios  
- Day.js  
- i18next  
- OpenWeather API (via Vercel Serverless Function)
